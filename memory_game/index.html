Instructions

Concentration, also known as Match Match, Memory Pelmanism, Shinkei-suijaku, Pexeso or simply Pairs, is a card concentration game in which all of the cards are laid face down on a surface and two cards are flipped face over each turn. The object of the card game is to turn over pairs of matching cards.

About the Game

The Memory Game is a game that has been developed independantly and played for centuries throughout many different societies and cultures. There is something inherently appealing about the both random and memory aspects of the game. Multiplayer variants of the game are also common and introduce an aspect of speed. Playing the Memory Game once a day is a fun way to keep your mind sharp!
